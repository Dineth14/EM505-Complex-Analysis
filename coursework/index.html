Coursework

A collection of homework solutions, project write-ups, and exploratory work produced during the complex analysis course. Each entry links to a PDF and summarises the topics covered.

Submitted Work

How to Add New Coursework

  1. Copy the template folder: duplicate coursework/template/ and rename it (e.g., coursework/hw3/).
  2. Edit data.json: fill in the title, date, topics list, and description.
  3. Add your PDF: place solution.pdf (or any name) in the same folder and update the "pdf" field.
  4. Register the entry: add a new object to the ENTRIES array in the <script> block at the bottom of this page.

Template data.json

{
  "title": "Homework N — Topic Title",
  "date": "2026-MM-DD",
  "topics": ["Topic A", "Topic B"],
  "description": "Brief summary of the problems solved.",
  "pdf": "solution.pdf"
}
